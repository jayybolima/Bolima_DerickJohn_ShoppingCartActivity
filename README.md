# Bolima_DerickJohn_ShoppingCartActivity

Shopping Cart System (C#)

Features
- Product menu with stock
- Input validation using int.TryParse()
- Prevents duplicate cart entries
- Stock deduction system
- Cart size limit
- Receipt with total and discount
- 10% discount if total ≥ 5000

Files
- Program.cs
- Flowchart.png
- README.md

How to Run
1. Open in Visual Studio or VS Code
2. Compile and run Program.cs
